# 💫 About Me:
🛠️ I specialize in the JavaScript/TypeScript ecosystem, building frontends with Next.js, Tailwind CSS, and Framer Motion 🎨 , and backends with HonoJS, Express.js, and Prisma ⚙️. My workflow involves strictly typed environments using TypeScript and monorepo structures with Nx 📦. I have practical experience implementing secure authentication (Clerk) 🔐 , payment gateways (Stripe Connect) 💳 , and database management with PostgreSQL, MongoDB, and Redis 🗄️.
